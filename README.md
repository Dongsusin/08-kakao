React를 이용하여 만들어본 kakao 클론사이트 입니다.

  <img src="./kakao.png"/>

## 컴포넌트 구성

- header(헤더 컴포넌트)
- visual(무한 스크롤 관리 컴포넌트)
- news(새로운 소식을 다루는 컴포넌트)
- banner(공식채널 연결 컴포넌트)
- business(회사의 중요시사항 컴포넌트)
- footer(바닥글 컴포넌트)

<br>

## 주요 기능

- 무한 스크롤 버튼 슬라이더
- 각 컴포넌트별 반응형 구현
- 각 뉴스의 태그 관리
